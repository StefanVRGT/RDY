# Background Video

Place your background video here as `background.mp4`.

Recommended specs:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 or higher
- Duration: 30-60 seconds (will loop)
- File size: Under 10MB for optimal loading
- Content: Slow, calming motion (nature, abstract patterns, gentle movements)

For testing, you can use a free stock video from:
- Pexels Videos: https://www.pexels.com/videos/
- Pixabay: https://pixabay.com/videos/
- Videvo: https://www.videvo.net/

Example search terms: "meditation", "calm nature", "slow motion water", "twilight sky"
